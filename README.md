# Base de datos ClimAPP

[![N|Solid](https://nodejs.org/static/images/logo.svg)](https://nodejs.org/es/)



Tecnologias implementadas:
* Nodejs
* Express
* MongoDB


#  -- Instalacion --   

* Clonar repositorio de GitHub
* Instalar dependencias:

    * NodeJs
    ```bsh
        npm install /
    ```
* Para iniciar el sistema  utilizar 
   
    ```bash
        nodemon
    ```

* Dependencies
    ```
        body-parser
        cookie-parser
        cors
        debug
        dotenv
        ejs
        express
        http-errors
        mongoose
        morgan
        nodemon
    ```
